RC Filters Collection

The RC Filters Collection is a set of educational circuit boards designed to support first- and second-semester electronics courses focused on analog circuit design and filter operation.

This collection includes eight foundational RC filter configurations:

  • Low Pass Active Filter
  • High Pass Active Filter
  • Band Pass Active Filter
  • Low Pass Active Filter
  • Active Notch Filter
  • Single Feedback Low Pass
  • Single Feedback High Pass
  • Single Feedback Band Pass

Each circuit is implemented as a standalone miniPCB™ with standardized layout, clear labeling, and accessible test points for easy integration into lab environments. These boards are ideal for reinforcing key learning outcomes such as feedback principles, waveform generation, frequency control, and timing behavior in analog circuits.

Instructors can use the full set to guide students through progressive exploration—from basic two-transistor oscillators to more advanced op-amp and feedback-based designs—while observing waveform behavior, frequency control, and stability through real-world measurements.
