04B-005 – Common Emitter Amplifier

A foundational BJT amplifier for teaching gain, biasing, and small-signal response.

Title: Common Emitter Amplifier

Board Size: Approx. 50 x 50 mm

Pieces per Panel: 4

Description

AI Seed

C1 and C3 are power rail decoupling capacitors. R1, R2 (trimmer), R3 set the bias point for the transistor base (gate). C2 AC couples the input signal, blocking the DC input signal. TP1 measures the input signal. TP2 measures the base (gate) of the transistor. TP5 and TP6 allow measurement of the resistor biasing network. C5, J2, and R10 form a feedback network that can be disconnected by J2. J2 is in series with C5 and R10. TP11 measures the node between C5 and R10. R5, R6, R9 (trimmer), C4, and J1 form a feedback and biasing network that can be bypassed by J1. J1 is in parallel with R5, R6, R9, and C4. TP7 and TP8 allow measurement of the RC feedback and biasing network. R4, R7 (trimmer), R8 form a resistor network on the transistor collector (drain). TP10 measures the transistor collector (drain). C6 AC couples the output signal, blocking the DC output signal. TP12 measures the output signal.

AI Generated

The 04B-005 is a configurable common-emitter amplifier PCB with defined bias, coupling, and feedback paths and comprehensive test access. C1 and C3 provide power rail decoupling. R1, R2 (trimmer), and R3 establish the DC operating point at the transistor base (gate), while C2 AC-couples the input to block any DC component.

Measurement is supported throughout: TP1 monitors the input signal; TP2 monitors the base (gate). TP5 and TP6 expose the bias divider nodes. On the output side, C6 AC-couples the amplified signal, and TP12 provides an output measurement point. The collector (drain) network uses R4, R7 (trimmer), and R8 to set load conditions and influence gain and headroom; TP10 taps the collector (drain) node.

Two reconfigurable feedback/bias subsystems are provided. C5, J2, and R10 form a series feedback path that can be interrupted with J2; TP11 measures the node between C5 and R10. A second RC feedback and biasing network uses R5, R6, R9 (trimmer), and C4; J1 is in parallel with this network to bypass it entirely. TP7 and TP8 provide observation points across the RC feedback/bias network, enabling analysis of frequency response and stabilization effects.

  • R2: trims base (gate) bias within the R1–R3 network.
  • R7: trims the collector (drain) network with R4 and R8.
  • R9: trims the RC feedback/bias network with R5, R6, and C4.
  • J1: bypasses the R5–R6–R9–C4 feedback/bias network.
  • J2: disconnects the C5–R10 series feedback path.
  • Key test points: TP1 (input), TP2 (base/gate), TP10 (collector/drain), TP12 (output).
