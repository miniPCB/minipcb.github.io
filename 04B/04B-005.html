04B-005 – Common Emitter Amplifier

A foundational BJT amplifier for teaching gain, biasing, and small-signal response.

Part No: 04B-005

Title: Common Emitter Amplifier

Board Size: Approx. 50 x 50 mm

Pieces per Panel: 4

Panel Size: 100 x 100 mm (V-scored)

Description

AI Seed

C1 and C3 are power rail decoupling capacitors. TP3 and TP4 measure V+ and GND respectively. R1, R2 (trimmer), R3 set the bias point for the transistor base (gate). C2 AC couples the input signal, blocking the DC input signal. TP1 measures the input signal. TP2 measures the base (gate) of the transistor. TP5 and TP6 allow measurement of the resistor biasing network. C5, J2, and R10 form a feedback network that can be disconnected by J2. J2 is in series with C5 and R10. TP11 measures the node between C5 and R10. R5, R6, R9 (trimmer), C4, and J1 form a feedback and biasing network that can be bypassed by J1. J1 is in parallel with R5, R6, R9, and C4. TP7 and TP8 allow measurement of the RC feedback and biasing network. R4, R7 (trimmer), R8 form a resistor network on the transistor collector (drain). TP10 measures the transistor collector (drain). C6 AC couples the output signal, blocking the DC output signal. TP12 measures the output signal.

AI Generated

The 04B-005 is a configurable single-stage common-emitter amplifier board with AC-coupled I/O, optional feedback paths, and comprehensive test-point access for measurement and tuning. On-board rail decoupling (C1, C3) stabilizes the supply; TP3 and TP4 expose V+ and GND for quick verification and current probing.

Input biasing is set by R1, R2 (trimmer), and R3, establishing the transistor base operating point. C2 AC couples the input to block DC; TP1 monitors the driven input while TP2 reads the base node. TP5 and TP6 access the bias divider nodes. An RC feedback/bias network composed of R5, R6, R9 (trimmer), and C4 can be enabled for local feedback or bypassed entirely with J1; J1 shorts across the network when installed. TP7 and TP8 instrument this network for gain/linearity studies.

The collector network (R4, R7 trimmer, R8) sets load and quiescent collector voltage; TP10 measures the collector for DC setup and small-signal gain checks. A second feedback path formed by C5 and R10 is placed in series with J2; opening J2 disconnects this path, while closing it engages frequency-selective feedback. TP11 observes the junction of C5 and R10. The output is AC coupled by C6 to remove DC offset; TP12 provides the output measurement point.

  • Adjustable base bias (R2) and collector setpoint (R7).
  • Selectable RC feedback via J1; frequency feedback via J2.
  • AC-coupled input/output (C2, C6) for signal integrity.
  • Extensive test points: input, base, bias nodes, collector, output.
