AI Generated

04B-005 implements a single‑stage common‑emitter topology with AC‑coupled I/O. C1 and C3 decouple the power rail; TP3 and TP4 read V+ and GND. C2 AC‑couples the input (TP1), blocking DC into the bias network. The base (gate) node is monitored at TP2. Output is AC‑coupled by C6 to TP12, isolating the stage’s DC operating point.

R1, R2 (trimmer), and R3 form a divider that biases the transistor base (gate); TP5 and TP6 probe this network. Collector (drain) loading and headroom are set by R4, R7 (trimmer), and R8; the collector (drain) is available at TP10. An emitter‑side feedback/bias network using R5, R6, R9 (trimmer), and C4 can be bypassed by shorting J1 (J1 is in parallel with the network); TP7 and TP8 observe this RC network. A secondary feedback path (R10 in series with C5) can be inserted with J2; TP11 reads the C5–R10 junction.

  • Bias divider R1/R3: 22–220 kΩ; lower values stiffen bias (better PSRR, higher quiescent draw), higher values reduce current but increase sensitivity to transistor β and noise. R2: 10–100 kΩ trim.
  • C1: 100 nF ceramic; C3: 10–100 μF bulk. Larger C improves supply rejection and transient headroom.
  • C2: 10 nF–1 μF; choose for f_c ≈ 1/(2π·R_in·C2). Larger C improves low‑frequency response at the cost of turn‑on transients.
  • Collector load (R4+R7+R8): 1–15 kΩ. Larger values raise gain but reduce swing/headroom; smaller values increase linearity and bandwidth.
  • Emitter network (R5/R6: 100–1.5 kΩ; R9: 500 Ω–5 kΩ trim; C4: 1–220 μF). More degeneration (higher R) lowers gain and distortion; C4 restores AC gain below f_c.
  • J2 feedback (R10: 10–220 kΩ; C5: 47 pF–1 nF): increases HF stability/roll‑off; larger C5 tames oscillation but reduces bandwidth.
  • C6: 1–100 μF; set for f_c ≈ 1/(2π·R_load·C6). Larger C extends bass into lower loads.

Set R2 for mid‑supply collector (TP10) at quiescence. Trim R7/R9 for desired gain and headroom, observing clipping at TP12. Use J1 open for stabilized, lower‑gain operation; close J1 for maximum gain. Enable J2 if peaking or oscillation is observed at high frequencies (TP11/TP12).
