miniPCB Tools

Use these tools to generate documents, analyze designs, decode part numbers, and navigate the board library.

Test Base 2026

Test Base 2026 is a structured document generator designed for engineering test planning. It creates and edits plan sections through repeatable templates and workflow controls. It centralizes review-ready content for test plans and related engineering documents.

Engineering Analyzer

Engineering Analyzer processes design-side JSON inputs such as netlist and FMEA setup data. It lets you configure analysis runs, execute calculations, and inspect structured outputs in one interface. It supports saved result workflows for iterative post-design analysis.

Part Number Radar

Part Number Radar is a visual web tool for constructing and decoding electronic component part numbers. It guides selection by family, option codes, and valid combinations so entries stay structured. It helps translate between readable requirements and vendor-style part-number strings.

Catalog Reader

Catalog Reader provides full-screen viewing for board markdown, JSON, and FMEA artifacts. It links content panels so you can inspect engineering text and structured data side by side. It gives fast read access to published board files without leaving the browser.

Catalog Browser

Catalog Browser is a numerical graphical interface for browsing the miniPCB catalog index. It supports filtering, sorting, and quick-open actions across board and collection pages. It is optimized for fast navigation when you already know target families or part numbers.

Board Finder

Board Finder helps locate existing miniPCBs by circuit topic, function, and learning intent. It combines guided prompts and search-style recommendations to narrow candidate boards quickly. It is an efficient entry point when you know the behavior you need but not the board number.
